# Introduction to AWC Lambda
 - What is AWS Lambda?
     AWS Lambda is a compute service that runs code without the need to manage servers. Your code runs, scaling up and down automatically, with pay-per-use pricing     
 - Why is Lambda useful ?
     - No Servers to manage.
     - Pay for what you use.
     - Automatic scaling.
     - Native service integration.
     - Highly available AWS infrastructure.
     - Built-in Secure isolation.
     - Supports wide range of compliance.
     - Highly performant. 
 - Disadvantages of AWS Lambda ?
    - With AWS Lambda you gain flexibility at the expense of control.
    - No visibility to internals
 - Who uses AWS Lambda ?
    - Batch data processing tasks
    - Real-time data processing
    - Generative AI applications, enabling you to start small and scale seamlessly while handling distributed, event-driven workflows securely at scale.
